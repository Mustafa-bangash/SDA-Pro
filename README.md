# SDA-Pro — Security Incident Response & Threat Mitigation Platform

## Course
Software Design & Architecture — Semester Project

## Team
| Role | Student | Patterns |
|------|---------|----------|
| Threat Pipeline Engineer | Student A | Singleton, Composite, Factory Method, Chain of Responsibility, Strategy, Abstract Factory, State |
| Integration & Response Engineer | Student B | Adapter, Decorator, Proxy, Facade, Chain of Responsibility |
| SOC Platform Engineer | Student C | MVC, Observer, SOA, Event-Driven |

## Tech Stack
- Backend: Java (Spring Boot)
- Frontend: React
- Message Broker: RabbitMQ
- Database: PostgreSQL + Redis
- Containerization: Docker + Docker Compose

## How to Run
docker-compose up
